# crm_2026

CRM Project containing:
- Frontend (Next.js)
- Backend (FastAPI + Supabase)
- Project Blueprint / Documentation

## Getting Started

### Frontend
1. cd frontend
2. npm install
3. npm run dev

### Backend
1. cd backend
2. pip install -r requirements.txt
3. uvicorn main:app --reload
